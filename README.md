**Codekart API文档 使用手册**

[Codekart](http://codekart.jojoin.com/) 是一套给 Node.js 开发者使用的应用程序开发框架和工具包。 它提供一套丰富的标准库以及简单的接口和逻辑结构， 其目的是使开发人员更快速地进行项目开发。 使用Codekart可以减少代码的编写量， 并将你的精力投入到项目的创造性开发上。
